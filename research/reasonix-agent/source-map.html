注意事项

  • docs/ARCHITECTURE.md 的 non-goals 中提到 RAG/vector retrieval 不是目标，但当前代码已有 src/index/semantic/* 和 semantic search 工具。这里应理解为文档与实现演进存在漂移。
  • src/tools/subagent.ts 文件头注释写了 non-streaming child loop，但后续构造 child CacheFirstLoop 时配置为 stream: true。这也是小范围注释漂移。
  • SECURITY 文档把 run_command 称为 shell sandbox，但从源码看它是命令解析、确认和路径限制，不是 kernel sandbox。
